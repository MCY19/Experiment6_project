이 pdf 파일들은 IT종합설계프로젝트 수업 프로젝트 과제에 사용된 코드 및 결과들을 첨부한 파일입니다. 

1. Augmentation_code: 이미지 데이터를 증강 작업할 때 사용한 코드로 
원본 이미지에 대한 좌우반전 및 밝기 조절을 할 수 있도록 만든 코드입니다. 

2. Crawling_code: 구글 이미지 사이트에서 자동으로 이미지를 검색하고 다운로드 받을 수 있게 하는 코드입니다.
input_element.send_keys에 원하는 키워드를 입력하고 코드를 돌리면 됩니다. 
HTTPError가 발생하는 것은 구글 이미지 사이트에 있는 이미지중 손실된 이미지가 있는 경우 발생합니다. 
해당 에러 코드가 뜨기 전까지 이미지 다운로드가 가능합니다. 

3. CNN_code: 수업시간에 배운 3-layer CNN 구조로 학습시킨 코드 및 결과를 첨부했습니다. 

4. Resnet_code: Resnet-50 모델을 활용하여 새로 학습시킨 코드 및 결과를 첨부했습니다. 

5. Model_Test: 이미지 모델을 테스트 할 수 있도록 고안된 코드입니다. 
인공지능 모델이 있으면 모델에 대한 이미지 분류 작업이 잘 되었는지 확인할 수 있습니다.
